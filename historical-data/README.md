# Historical season packs

Cada archivo `AÑO.json` contiene equipos, corredores, ratings, fuente y estado de completitud de una temporada.

Packs incluidos:
- `2018.json`
- `2026.json`

El archivo `manifest.json` controla qué años aparecen como disponibles en la interfaz. No cambies el manifiesto manualmente: el generador lo actualiza junto con `historical-manifest.js`.
