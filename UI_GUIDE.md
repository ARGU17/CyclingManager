## Componentes v0.29

- `v029-kpi-grid`: indicadores financieros y estructurales.
- `v029-campus-grid`: plano visual de instalaciones.
- `v029-facility-card`: nivel, condición, capacidad, coste y proyecto.
- `v029-sponsor-card`: contrato u oferta comercial.
- `v029-finance-grid`: proyección de ingresos y gastos.
- `v029-fan-bars`: mercados regionales.
- `v029-org-grid`: organigrama.
- `v029-base-card`: bases secundarias.
- `v029-event-alert`: decisiones dinámicas.

## Tipografía

Se usa exclusivamente la pila de fuentes del sistema. No se distribuyen archivos de tipografía:

```css
-apple-system,
BlinkMacSystemFont,
"SF Pro Display",
"SF Pro Text",
"Helvetica Neue",
Arial,
sans-serif
```

Los datos financieros y de telemetría conservan la pila monoespaciada del sistema.
