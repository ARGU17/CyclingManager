# Cycling Manager Tour v0.19

Versión v0.19 con objetivos de sponsor, mercado/contratos/cantera/scouting, planificación de forma/calendario individual y pantalla TV, conservando Race Director v0.17.

## Instalación

Sube `index.html`, `data.js`, `game.js` y `styles.css` a GitHub Pages. Al abrir, pulsa **Borrar guardado** para evitar estados antiguos.

## Cambios v0.19

- Objetivos dinámicos de sponsor por carrera.
- Evaluación de objetivos al final de cada carrera.
- Presupuesto, prestigio y confianza del sponsor.
- Mercado con contratos, renovaciones, rescisiones, fichajes, cantera y scouting regional.
- Staff técnico con efectos sobre entrenamiento, nutrición, material y scouting.
- Planificación de picos de forma por corredor: objetivo A/B/C, carrera objetivo y tipo de preparación.
- Training camps v0.19: Teide, Sierra Nevada, Andorra, Girona, Flandes/Roubaix, Mallorca, Aero Lab, calor, Dolomitas y recuperación médica.
- Vista TV con marcador, ticker, narración y datos de grupos.
- Perfil 2D mejorado con 1 punto/km, microgradientes, rejilla, bandas de puerto y barras de pendiente.
